Instructions to Execute

1. Unzip the rar file

2. Go to the IS_UltraSonic directory using the terminal or cmd

3. Run "./gradlew cleanEclipse eclipse" if your on linux or "gradlew cleanEclipse eclipse" if your on windows

4. Run "./gradlew build" or "gradlew build" to build the project

5. Run "./gradlew run" or "gradlew run" to run the project
